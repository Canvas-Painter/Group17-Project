Canvas Painter Help

Dark Mode

Toggle the dark mode switch to instantly apply a dark theme to Canvas.

Custom Themes

  1. Enter a theme name
  2. Select colors for backgrounds, text, sidebar, and other elements
  3. Click "Save Theme" to store your custom theme
  4. Use the dropdown to select and apply saved themes

Managing Themes

  • Edit Theme: Select a theme and modify its colors
  • Delete Theme: Remove unwanted themes from your saved list
  • Apply Theme: Toggle the switch to apply/remove the selected theme

Standard Syllabus

For quick access to a course's syllabus:

  1. Navigate to any page within your Canvas course
  2. Look for the "Standard Syllabus" link at the bottom of the sidebar
  3. Click the link to view the course syllabus directly

Note: This link is automatically added to every course page for easy access.

Weird due Date

Nothing special for this one. If an assignment has a due date different then midnight it will appear obvious.

Gradebook Required Grade

The gradebook page will have an additional menu where you can specify a custom assignment

  1. You input the category of the assignment. (If relevant)
  2. You input the points the assignment is worth
  3. You input the total grade in the class you want

And our software will tell you want you would need to get on the assignment to have that grade

Note: This does not take into account what-if grades you put in.
